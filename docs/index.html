Proyecto Boda - Sistema de Gestión en C++

Descripción

Este proyecto es un sistema de gestión de invitados, usuarios, recursos y mesas para un evento de boda, desarrollado en C++ usando listas dinámicas y memoria manual.

Repositorio GitHub

Puedes ver el código y la estructura completa del proyecto en mi GitHub: ProyectoBoda

Menú Principal

El sistema cuenta con un menú interactivo en consola que permite gestionar invitados, usuarios, recursos y mesas.
